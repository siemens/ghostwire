Ghostwire Classic API (1.0.1)

Download OpenAPI specification:Download

Returns the discovered network namespaces, the (virtual) network toplogy, the containers and processes using these network namespaces, and their communication configuration, including DNS client settings.

query Parameters
ieappicons
string
Enum: "" "on" "off"

Optionally request the discovery of Industrial Edge App icons.
